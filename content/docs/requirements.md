##### .Net Library
Journey .Net library requires .Net 8.0 and above.

##### Journey Command
Journey Command CLI requires windows/linux 64bit version.

##### Supported Databases
| Database | Version |
| :------------- | :------------ |
| Postgres       | 9.0+          |
| Mysql          | 5.0+          |
| Sql Server     | 2008+         |
| Sqlite         | 3             |
| Cockroach Db   | 2.0+          |
| Cassandra      | 1.2+          |
| Timescale Db   | 2.0+          |
